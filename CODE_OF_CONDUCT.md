# Code of Conduct

This account follows the Contributor Covenant Code of Conduct, version 2.1, for public collaboration.

## Expected behavior

- Communicate with technical accuracy and respect for scope.
- Provide reproducible reports, examples, and failure cases when possible.
- Keep discussions focused on the repository, issue, or pull request.
- Disclose conflicts of interest when they affect technical decisions.

## Unacceptable behavior

- Harassment, abuse, threats, or discriminatory conduct.
- Publishing private information without explicit permission.
- Deliberately disruptive comments, spam, or off-topic campaigning.
- Submitting fabricated security reports, benchmarks, or evidence.

## Enforcement

Repository maintainers may remove comments, lock threads, close issues, block users, or report abuse when behavior prevents productive maintenance.

## Contact

For conduct concerns, use the repository security contact path or open a concise private report when available.
